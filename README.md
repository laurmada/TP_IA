# Escolha Otimizada de Time Pokémon utilizando Algoritmo Genético e Satisfação de Restrições

## Sobre o Projeto

Este projeto tem como objetivo desenvolver um sistema inteligente para realizar a escolha de um time de **6 Pokémon dentre os 151 Pokémon da primeira geração** para enfrentar os **8 ginásios da região de Kanto**.

A escolha do time é realizada utilizando técnicas de **Inteligência Artificial**, combinando:

- **Algoritmo Genético:** utilizado para buscar uma combinação otimizada de Pokémon, simulando um processo evolutivo de seleção, cruzamento e mutação de indivíduos.
- **Satisfação de Restrições:** utilizada para considerar regras e restrições relacionadas aos tipos dos Pokémon, vantagens e desvantagens contra os líderes de ginásio.

O objetivo é encontrar uma equipe equilibrada capaz de obter o melhor desempenho possível contra todos os ginásios, considerando fatores como:

- Tipos dos Pokémon;
- Vantagens e fraquezas de cada tipo;
- Compatibilidade do time contra os desafios;
- Poder ofensivo e defensivo da equipe.
